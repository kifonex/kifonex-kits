# Kifonex Kits | کیت‌های معرفی کیفونکس

این مخزن شامل متن‌های آماده برای معرفی بخش‌های مهم کیفونکس است؛ از معاملات P2P ایرانی با تومان، ریال و لیر گرفته تا درگاه پرداخت کریپتو برای فروشگاه‌ها، ووکامرس، WHMCS و تلگرام.

هدف این مجموعه این است که هر قابلیت مهم کیفونکس در یک صفحه جداگانه، ساده و قابل فهم معرفی شود تا کاربران، تاجران و کسب‌وکارها سریع‌تر مسیر مناسب خود را پیدا کنند.

## محورهای اصلی کیفونکس

* معاملات P2P تتر با تومان، ریال و لیر
* ثبت آگهی خرید و فروش تتر
* مدیریت مرحله‌ای سفارش‌ها
* درگاه پرداخت کریپتو برای فروشگاه‌ها
* پرداخت رمزارزی برای WooCommerce
* پرداخت رمزارزی برای WHMCS
* لینک پرداخت کریپتو برای فروشندگان تلگرام
* احراز هویت و کنترل معاملات حساس
* فاکتور پرداخت قابل پیگیری برای کسب‌وکارها
